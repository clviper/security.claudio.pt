APKS Filesystem Locations

Tue Jul 28, 2015 by

User installed applications apks are stored at /data/app/ but there are few more locations where other types of apks reside, such as vendor apks that come with the phone and system permissions apks.

Those can be stored on the following locations:

  • /system/app/
  • /system/vendor/
  • /system/priv-app/ (system permission since 4.4)
